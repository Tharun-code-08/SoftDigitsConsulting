* Frontend: HTML / CSS / JavaScript *(or React / Next.js if applicable)*

---

## 📦 Installation & Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```

2. Navigate to project folder:

   ```bash
   cd your-repo-name
   ```

3. Install dependencies (if using React/Next.js):

   ```bash
   npm install
   ```

4. Run locally:

   ```bash
   npm run dev
   ```

---

## 🌍 Deployment

The website is deployed using:

* Vercel / Netlify

To deploy:

1. Push code to GitHub
2. Connect repository to Vercel/Netlify
3. Add custom domain: **softdigitconsulting.com**
